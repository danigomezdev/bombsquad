# Replay Mod for BombSquad

An advanced replay system that transforms how you experience BombSquad matches. Go beyond basic playback with professional-grade controls, cinematic camera tools, and real-time analysis that brings your replays to life like never before.

---

## Overview

This comprehensive mod completely reimagines BombSquad's replay functionality, turning simple playback into an immersive viewing experience. Whether you're analyzing competitive matches, creating cinematic content, or reliving epic moments, this toolkit provides everything you need with unprecedented control and flexibility.

Watch your matches from any angle, at any speed, with broadcast-quality controls that put you in the director's chair of your own BombSquad productions.


---

## Features

### Core Functionality
- **Instant Replay Loading** - Start watching immediately without waiting for file analysis
- **Smart Duration Scanning** - Background scanning that estimates duration while you watch
- **Multiple Replay Support** - Browse and select from all your replay files

### Playback Controls
- **Variable Speed** - Adjust playback speed from snail mode to quake pro
- **Precise Seeking** - Jump to any point in the replay with progress bar or timed seeks
- **Pause/Resume** - Full control over playback with visual indicators
- **Restart Function** - Quickly restart from the beginning

### Camera System
- **Free Camera Movement** - Look anywhere by dragging or using cinema mode
- **Manual Zoom** - Adjust zoom levels with precision controls
- **Player Tracking** - Auto-follow players or set custom look points
- **Cinema Mode** - Hide UI for cinematic viewing experience

### User Interface
- **Clean Overlay** - Non-intrusive control bar that can be hidden
- **Real-time Progress** - Visual progress bar with time displays
- **Smart Buffering** - Know exactly how much replay data is loaded
- **Contextual Messages** - Helpful feedback for all actions

### Advanced Features
- **File Management** - Copy replay paths or show file locations
- **Error Handling** - Graceful handling of corrupted or incomplete files
- **Background Processing** - Heavy operations don't block the UI
- **Memory Efficient** - Streams data without loading entire files

## Installation

1. Download the latest version of the mod
2. Place the file in your BombSquad `mods` folder
3. Restart the game

**Requirements:**
- BombSquad 1.7.51 or later
- API 9 compatibility

---

## Usage

### Accessing Replays

1. **In-Game Menu**: Click the Replay button in the pause menu during a game
2. **Watch Window**: Access from the watch window interface

### Basic Controls

- **Play**: Select a replay and click Run
- **Pause/Resume**: Use the pause button or spacebar
- **Seek**: Click on the progress bar or use seek buttons
- **Speed Control**: Adjust playback speed with fast-forward/rewind buttons
- **Hide UI**: Use the hide button for unobstructed viewing

### Camera Controls

- **Free Look**: Enter cinema mode and drag to look around
- **Zoom**: Use +/- buttons or scroll wheel
- **Reset**: Return to automatic player tracking
- **Save Position**: Save custom camera angles

### File Operations

- **Show Path**: Display the full file path of a replay
- **Copy Path**: Copy the file path to clipboard for sharing
- **Auto-scan**: Let the mod automatically analyze replay duration

## Technical Details

### Replay Format
- Uses BombSquad's native `.brp` replay format
- Implements Huffman compression for efficient storage
- Supports both compressed and uncompressed replay data

### Performance
- Background thread processing for large files
- Progressive loading to avoid memory issues
- Real-time decompression during playback

### Compatibility
- Works with all official BombSquad game modes
- Compatible with most other mods
- Supports both single and multiplayer replays

## Troubleshooting

### Common Issues

**Replay won't play:**
- Ensure the replay file isn't corrupted
- Check that you have sufficient memory
- Verify game version compatibility

**Camera controls not working:**
- Make sure you're in manual camera mode
- Check that cinema mode is properly activated

**Performance issues:**
- Close other applications to free up memory
- Reduce replay quality settings if available

### Error Messages

- **"Select a replay!"**: No replay file is selected
- **"A replay is already running!"**: Wait for current replay to finish
- **"Couldn't load replay!"**: File may be corrupted or incompatible
