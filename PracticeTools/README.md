# Practice Tools for BombSquad

A comprehensive mod that provides advanced training and practice utilities for BombSquad players.

## Description

Practice Tools transforms your BombSquad experience into a full-featured training environment. This mod allows players to spawn bots, power-ups, and access various training utilities to improve their skills in a controlled practice setting.

## Features

- **Bot Spawning System**: Spawn various types of bots with customizable counts and spawn radius
- **Power-up Management**: Spawn any power-up type or clear all active power-ups
- **Visual Aids**: Display bomb countdown timers and radius visualizers
- **Training Utilities**: Invincibility mode, bot freezing, and dummy immortality options
- **Real-time Configuration**: Toggle features on-the-fly without restarting
- **Multiplayer Support**: Works in local multiplayer sessions
- **Customizable Settings**: Extensive configuration options for personalized training

## Installation

1. Download the `practice_tools.py` file
2. Place it in your BombSquad mods folder
3. Ensure you have BombSquad version with API 9 support

## Usage

### Accessing Practice Tools
1. Launch BombSquad with the mod installed
2. From the main menu, click the "Practice" button in the party window
3. Use the tabbed interface to access different tool categories

### Available Tabs

#### Bots Tab
- **Bot Selection**: Choose from 18 different bot types including Dummy, Bomber, Brawler, Trigger, Charger, and Pro variants
- **Count Control**: Spawn 1-10 bots at once
- **Spawn Radius**: Small, Medium, or Large spawn areas
- **Bot Controls**: Stop bot movement or make dummies immortal

#### Power-ups Tab
- **Power-up Selection**: Spawn any power-up including bombs, curses, health, shields, and special abilities
- **Debuff Function**: Remove all active power-ups from players
- **Visual Options**: Toggle bomb countdowns and radius visualizers
- **Power-up Behavior**: Control whether power-ups expire normally

#### Others Tab
- **Game Controls**: Pause the game while menu is open
- **Player Options**: Enable invincibility for all players
- **Game Mode**: Toggle epic mode (slow motion)

## Bot Types Available

### Basic Bots:
- Dummy, Bomber Lite, Brawler Lite
- Bomber, Brawler, Trigger, Charger
- Sticky, Explodey, Bouncy

### Pro Bots:
- Pro Bomber, Pro Brawler, Pro Trigger, Pro Charger
- Shielded Pro variants with enhanced capabilities

## Power-up Types

- Triple Bombs, Curse, Health
- Ice Bombs, Impact Bombs, Land Mines
- Punch, Shield, Sticky Bombs

## Technical Features

- **Advanced Bot AI**: Modified bot behavior with customizable movement
- **Real-time Visual Updates**: Dynamic radius and countdown displays
- **Config Persistence**: Settings saved between game sessions
- **Multiplayer Sync**: Supports multiple players in practice sessions
- **Performance Optimized**: Efficient implementation with minimal performance impact

## Configuration

The mod automatically creates configuration entries for:
- Practice Tab preferences
- Bomb countdown displays
- Radius visualization toggles
- Bot behavior controls
- Power-up expiration settings
- Player invincibility options

## Compatibility

- Requires BombSquad with API 9 support (build 20427 or newer)
- Compatible with most other mods
- Works in local multiplayer games
- No conflicts with existing game mechanics

## System Requirements

- BombSquad version with API 9 support
- Python modding environment
- Sufficient system resources for additional bot AI processing

## Known Issues

- Practice menu may not function in network games
- Some visual effects may not appear on lower-end devices
- Bot count limitations may apply on older hardware

## Troubleshooting

- Ensure you have the correct BombSquad version
- Verify the mod file is in the correct mods folder
- Check that no other mods are causing conflicts
- Restart the game if features don't appear initially
